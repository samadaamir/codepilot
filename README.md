# 🚀 CodePilot

CodePilot is an AI-powered code review platform that helps developers write cleaner, more efficient, and maintainable code. Users can write or paste their source code into an interactive editor, and the AI analyzes it to provide code reviews, improvements, bug fixes, optimization suggestions, and best practice recommendations.

---

## 📌 Features

* ✨ **AI Code Review**

  * Analyze code using the Gemini API.
  * Detect potential bugs and issues.
  * Receive detailed explanations and suggestions.

* 💡 **Code Improvements**

  * Get recommendations for cleaner and more efficient code.
  * Improve readability and maintainability.
  * Follow modern coding best practices.

* 🛠 **Code Corrections**

  * Identify syntax and logical errors.
  * Receive corrected code with explanations.

* ⚡ **Interactive Code Editor**

  * Built with **Monaco Editor** (VS Code editor experience).
  * Supports writing or pasting code directly.
  * Syntax highlighting for an enhanced coding experience.

* 🎨 **Modern Responsive UI**

  * Fully responsive design using Tailwind CSS.
  * Clean and intuitive user interface.
  * Optimized for desktop and mobile devices.

* 📋 **Features Page**

  * Dedicated page showcasing all platform features.
  * Interactive feature cards with modern UI.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/codepilot.git
```

Navigate to the project directory:

```bash
cd codepilot
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add your Gemini API key:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

---

## 🚀 Usage

1. Open the application.
2. Write or paste your source code into the Monaco Editor.
3. Click the **Review Code** button.
4. The AI analyzes your code using the Gemini API.
5. Review the generated:

   * Code improvements
   * Bug detection
   * Corrected code
   * Performance suggestions
   * Best practice recommendations
